Расшифровывает файл с использованием AES-256-ECB.

Функция извлекает соль из зашифрованного файла, использует её для генерации ключа на основе пароля пользователя с помощью PBKDF2, а затем расшифровывает данные, записанные в файл.

Аргументы
filePathПуть к файлу, который нужно расшифровать.
passwordПароль для генерации ключа шифрования.
parentУказатель на родительский объект (по умолчанию nullptr).
Возвращает
true, если расшифровка прошла успешно.
false, если произошла ошибка (например, неверный пароль или повреждённые данные).
Заметки
Функция использует AES-256-ECB без IV. В случае повреждённых данных или неверного пароля функция вернёт false.
Аргументы
filePathПуть к файлу, который нужно расшифровать.
passwordПароль для генерации ключа шифрования.
parentУказатель на родительский объект (по умолчанию nullptr).
Возвращает
true, если расшифровка прошла успешно.
false, если произошла ошибка (например, неверный пароль или повреждённые данные).

bool encryptFile ( const QString & filePath,

Шифрует файл с использованием AES-256-ECB.

Функция генерирует случайную соль для шифрования, использует её для генерации ключа на основе пароля пользователя с использованием PBKDF2, а затем шифрует файл с использованием алгоритма AES-256-ECB. Зашифрованные данные записываются в файл.

Аргументы
filePathПуть к файлу, который нужно зашифровать.
passwordПароль для генерации ключа шифрования.
parentУказатель на родительский объект (по умолчанию nullptr).
Возвращает
true, если шифрование прошло успешно.
false, если произошла ошибка (например, не удалось открыть файл или сгенерировать ключ).
Заметки
Используется алгоритм AES-256-ECB без IV. Этот режим шифрования не обеспечивает защиту от атак на повторение блоков, поэтому рекомендуется использовать его только для простых случаев.
Аргументы
filePathПуть к файлу, который нужно зашифровать.
passwordПароль для генерации ключа шифрования.
parentУказатель на родительский объект (по умолчанию nullptr).
Возвращает
true, если шифрование прошло успешно.
false, если произошла ошибка (например, не удалось открыть файл или сгенерировать ключ).
